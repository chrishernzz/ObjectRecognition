Step 1: python -m venv yoloenv (create the virtual environment)

Step 2: yoloenv\Scripts\activate (this will allow you to use the yoloevn)

Step 3: pip install Ultralytics (installed the libraries that is needed to use YOLO)
        Note: may need to install the upgrade version

Step 4: python main.py (run this in the terminal once you done with Step 3)
